The following tools are available:

info

Inspection of version and configuration information.

logger

Console-based realtime logging of events.

introspect

Console-based introspection of running RSB systems.

send

Commandline-based sending of events.

call

Commandline-based calling of RPC server methods.

timesync

Synchronization of events on different scopes based on timestamps.

bridge

Forwarding of events from one RSB bus to another.

server

A standalone server for the RSB socket transport.

web

Web-based interface to information about a running RSB system such as introspection. Also provides multiple HTTP-endpoints for retrieving the information programatically.
