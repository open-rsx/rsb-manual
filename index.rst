.. _rsb:

Welcome to the |project| Manual! (for version |version|)
========================================================

The Robotics Service Bus (|project|) is a message-oriented,
event-driven middleware aiming at scalable integration of robotics
systems in diverse environments.

.. note::

   Starting with this version of |project|, support for the
   :term:`Spread` :term:`transport` is no longer automatically
   included/installed with |project|.

   .. seealso::

      :ref:`install-debian`
        For instructions on installing support for :term:`Spread`
        using Debian packages.

      :ref:`troubleshooting-spread-does-not-work`
        For more instructions on getting the :term:`Spread`
        :term:`transport` to work.

      :ref:`news`
        For a detailed list of changes in this version.
