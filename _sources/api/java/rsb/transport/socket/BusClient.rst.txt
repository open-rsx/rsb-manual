BusClient
=========

.. java:package:: rsb.transport.socket
   :noindex:

.. java:type:: public class BusClient extends BusBase

   A \ :java:ref:`Bus`\  implementation which acts as a client to an existing server implementation. This means that new notifications are only sent to local receivers as well as to a single \ :java:ref:`BusConnection`\  connecting with the server.

   :author: swrede, jwienke
