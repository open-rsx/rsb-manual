RouteConfiguratorUtility
========================

.. java:package:: rsb.eventprocessing
   :noindex:

.. java:type:: public class RouteConfiguratorUtility<ConnectorType extends Connector>

   A utility class that can be used via composition to implement some of the methods required by the \ :java:ref:`RouteConfigurator`\  interface.

   :author: jwienke
   :param <ConnectorType>: type of connectors
