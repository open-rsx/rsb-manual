CachedCharsetCoding
===================

.. java:package:: rsb.converter
   :noindex:

.. java:type:: public class CachedCharsetCoding

   Lazily creates and caches a \ :java:ref:`java.nio.CharsetEncoder`\ , \ :java:ref:`java.nio.CharsetDecoder`\  pair for each calling thread.

   :author: jmoringe

Constructors
------------
CachedCharsetCoding
^^^^^^^^^^^^^^^^^^^

.. java:constructor:: public CachedCharsetCoding(Charset charset)
   :outertype: CachedCharsetCoding

   Creates a per-thread cache for encoder-decoder pairs for the given character set.

   :param charset: The character set that should be used for en- and decoding.
