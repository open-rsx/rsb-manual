FilteringHandler
================

.. java:package:: rsb
   :noindex:

.. java:type:: public class FilteringHandler implements Handler

   A decorator for \ :java:ref:`Handler`\ s that allows additional filtering for each handler. Filters are a conjunction, hence one failure prevents delivery.

   :author: jwienke
