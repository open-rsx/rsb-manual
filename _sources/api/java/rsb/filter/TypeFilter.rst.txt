TypeFilter
==========

.. java:package:: rsb.filter
   :noindex:

.. java:type:: public class TypeFilter implements Filter

   A \ :java:ref:`Filter`\  that only accepts events which have a payload of a specific type. Events are accepted if their payload can be assigned to the desired class (ie. also subclasses are accepted).

   :author: swrede, jwienke
