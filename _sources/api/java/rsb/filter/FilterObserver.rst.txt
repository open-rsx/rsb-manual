FilterObserver
==============

.. java:package:: rsb.filter
   :noindex:

.. java:type:: public interface FilterObserver

   Interface for classes that want to be notified about changes to registered \ :java:ref:`Filter`\  instances.

   :author: jwienke, swrede

.. java:method::  void notify(Filter filter, FilterAction action)
