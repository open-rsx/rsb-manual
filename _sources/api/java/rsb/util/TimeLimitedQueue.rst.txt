TimeLimitedQueue
================

.. java:package:: rsb.util
   :noindex:

.. java:type:: public class TimeLimitedQueue extends LimitedQueue<Event>

   A time and space limited queue of RSB \ :java:ref:`Event`\ s that discards past events older than a specified time limit. It will additionally discard old events when reaching a set maximum capacity.

   :author: dklotz
