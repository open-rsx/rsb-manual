EventQueueAdapter
=================

.. java:package:: rsb.util
   :noindex:

.. java:type:: public class EventQueueAdapter extends AbstractEventHandler

   An adapter similar to the \ :java:ref:`QueueAdapter`\  that provides access to a queue of RSB \ :java:ref:`Event`\ s instead of directly to the payload data. It can also be directly registered as handler in an rsb.Listener instance and used for receiving and storing dispatched events.

   :author: dklotz

.. java:constructor:: public EventQueueAdapter(int capacity, boolean discardOldest)

   :param capacity: capacity of the internal queue
   :param discardOldest: if \ ``true``\ , remove older events if the queue is full, otherwise block until space is available on inserts
