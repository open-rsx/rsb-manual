LinuxHostInfo
=============

.. java:package:: rsb.util.os
   :noindex:

.. java:type:: public class LinuxHostInfo extends PortableHostInfo

   Linux-specific implementation of \ :java:ref:`CommonHostInfo`\  class. Tries to compute and cache a unique machine id (and hostname information) based on the following information:

   ..

   #. contents of /etc/machine-id
   #. contents of /var/lib/dbus/machine-id
   #. local hostname

   :author: swrede, ssharma, jwienke
