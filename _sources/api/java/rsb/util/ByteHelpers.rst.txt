byteBufferToArray
^^^^^^^^^^^^^^^^^

.. java:method:: public static byte[] byteBufferToArray(ByteBuffer buffer)
   :outertype: ByteHelpers

   Converts a \ :java:ref:`ByteBuffer`\  instance to a plain byte array. This method respects whether the buffer is read-only or has an underlying array available. In cases where an array is present and can be accessed, this array will be returned. Otherwise a copy is created.

   :param buffer: the buffer to convert
   :return: array with contents of the buffer. Should not be modified usually
