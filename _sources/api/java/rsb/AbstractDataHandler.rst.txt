AbstractDataHandler
===================

.. java:package:: rsb
   :noindex:

.. java:type:: public abstract class AbstractDataHandler<DataType> implements Handler

   A handler that receives the user payload of an event by extracting the data and casting them to the specified type.

   :author: swrede
   :param <DataType>: the desired target data type of the user handler. The event payload will be casted to this type
