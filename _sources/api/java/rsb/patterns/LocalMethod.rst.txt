LocalMethod
===========

.. java:package:: rsb.patterns
   :noindex:

.. java:type::  class LocalMethod extends Method implements Handler

   Objects of this class implement and make available methods of a local server. The actual behavior of methods is implemented by invoking arbitrary user-supplied functions.

   :author: jmoringe, swrede, jwienke
