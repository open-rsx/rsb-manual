Converter¶

public interface Converter<WireType>¶

This class represents a converter interface for a wire format T. Implementations may support one or more domain types for (de-)serialization to T and back to a specific object type referenced through the typeinfo parameter.

Author

swrede

Parameters

<WireType> – the wire format to serialize on
