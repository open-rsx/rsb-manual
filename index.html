Welcome to the RSB Manual! (for version 1.0)¶

The Robotics Service Bus (RSB) is a message-oriented, event-driven middleware aiming at scalable integration of robotics systems in diverse environments.

Note

This manual describes the development (or “master”) version of RSB.

When possible, the most recent stable version of RSB should be used instead. That version is described in a separate manual.
